Arch Linux Installation

A guide for installing Arch Linux, from scratch. Please read all the steps before asking for help.

Prerequisites:
  • Arch Linux ISO burned to a USB thumb drive.
  • Basic competency with the Linux command line interface.
  • An internet connection on the installation PC. Preferrably ethernet or wired.
  • At least 32 Gb free on a hard drive.
Whether or not you are dual booting off the same drive. If it is on the same drive, shrink your Windows partition by at least 32 Gb, making sure it is at the tail end of the disk.
Notes and warnings before we get started here:
  • This was written for a Desktop Computer, not a laptop. You are on your own if you have a dual graphics laptop. It's possible, but you cannot use the dedicated GPU.
  • This guide is built for a PC with an AMD GPU. This is because, out of the box, Arch does not like NVIDIA GPUs. I have an RX 580 8Gb on the system I installed on.
  • Commands in text will be in italics like so, to make them stand out.
If you have an Nvida GPU, go install Endeavour OS instead. I would also swap kernels as I dislike the mainline and prefer LTS myself.

Connecting to the internet

After plugging in your drive, and selecting it in your system's boot menu, we must first verify that you are connected to the internet. To do this, once all the text has gone away and you see "root@archiso" type in ping archlinux.org -c 4 to test internet connectivity. If you're wired in, this should work just fine, and you can carry on.
